# Kumo-2.5
<img src="https://github.com/kumoishibo/Kumo-2.5/blob/main/Images/KumoLogo.png?raw=true" width=30% alt="Kumo Logo" style="display: block;margin-left: auto; margin-right: auto;">

A 2.5" drone frame. Star shaped dual arm design available both in printable and CarbonFiber cut files in STL and STEP formats.

This project was born out of the desire for building a robust freestyle 2.5" 2S quad built from the ground up around the HDZero AIO15.
